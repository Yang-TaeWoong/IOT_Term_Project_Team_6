# IOT_Term_Project_Team_6
IOT 팀 6조 프로젝트입니다.

### 프로젝트 목표

* 스마트 IOT 장비를 이용한 산불 발생 및 전파 예방
* 실시간 데이터 수집을 통한 상황 감시
* 위치 데이터를 통한 산불 진행 시각화
* 카메라를 통한 해당 지역 감시
* 상활실에서의 빠른 도움
